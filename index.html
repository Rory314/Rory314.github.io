Roleplaying Overhaul Project


The Roleplaying Overhaul Project (ROP) aims to greatly enhance the community's roleplaying experience by creating a brand new game from scratch.
Expand all | Close all

A brand new game, inspired by the strategy games we know and love, will be created from scratch. Doing this will allow for

  • the creation of completely custom maps and an end to landswapping;
  • a comprehensive set of tools to allow game designers to draw and customise maps, specify starting units and resources (and more) and have their ideas automatically turned into new game rounds;
  • the addition of new mechanics, units, and resources that can all be customised to suit a particular game;
  • an end to tedious rule enforcement, rules will instead be specified for each round and handled automatically;
  • many more features to be decided by the community (see below).

Most importantly, the game will be created with roleplaying and the community in mind. This means no payments or unfair advantages, constant support for the community, and greatly expanded roleplaying features.

Whether you've designed your own rounds, have been around for years, or are just a newbie, we would love to hear from you about what you want to see.

A short (anonymous) survey has been created for you to give your views on the current state of roleplaying in S1914 & CoW, how you feel about roleplaying in general, and what you want to see from a project like this.

Please bear in mind that this project is likely to take a very long time to complete. I will not always have time (or motivation) to work on it.

Running the servers will be the hardest part, technically and financially. If you have any experience with this, don't hesitate to contact me.

For anyone wondering, the game is being developed in Java. Swing is used for the GUI and there is a simple test server running on my local machine.

Once the game is nearly ready for testing, I will look at releasing this on Android. Web and iOS versions will only be considered if there is significant support for this.
